Docker

TellForm provides an official Docker image via Dockerhub that can be used for deployments on any system that is running Docker. Hereâ€™s a one-liner that will start a container running TellForm:

$ git clone https://github.com/tellform/docker_files
$ cd docker_files
$ docker-compose up -d

This will launch a TellForm server on port 3000 by default. To access your TellForm server, navigate to http://localhost:3000 on your browser.
